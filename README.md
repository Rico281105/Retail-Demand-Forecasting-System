# Retail Demand Forecasting System

## Overview

A machine learning project that forecasts future retail sales using historical Superstore sales data.

## Technologies

- Python
- Pandas
- Prophet
- SQL
- Streamlit
- Matplotlib
- Seaborn

## Features

- Data Cleaning
- Exploratory Data Analysis
- Time-Series Forecasting
- Sales Trend Analysis
- Interactive Dashboard
- SQL Storage

## Run

python src/preprocess.py

python src/train.py

python src/forecast.py

streamlit run dashboard/app.py
